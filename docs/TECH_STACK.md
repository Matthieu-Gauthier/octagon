# Technology Stack - Frontend

## Core Framework
- **React**: Version 18+ (via Vite)
- **TypeScript**: For type safety and better developer experience.
- **Vite**: Fast build tool and development server.

## Styling & UI
- **Tailwind CSS**: Utility-first CSS framework for rapid styling.
- **Radix UI**: Headless UI primitives for accessible components (Dialog, Popover, Slot, etc.).
- **Lucide React**: Icon library.
- **Shadcn UI**: Re-usable component patterns built with Radix UI and Tailwind.
- **Sonner**: For toast notifications.

## State Management
- **Zustand**: Lightweight state management for handling bets and user data.
- **React Hook Form**: specific form handling and validation.
- **Zod**: Schema validation for form data.

## Routing
- **React Router DOM**: Client-side routing for navigation between Fight Card and Leaderboard.

## Development Tools
- **ESLint**: Code linting.
- **Prettier**: Code formatting.
