## Implementation Strategy

### Phase 1 — Backend (US1)
1. Add `rankingPosition Int?` and `isChampion Boolean?` to `Fighter` model in `schema.prisma`
2. Apply schema via `npx prisma db push`
3. Extend `ScrapedFighter` interface with new optional fields
4. Write TDD tests for the scraping logic (failing first)
5. Implement parsing in `scrapeFighter()`:
   - Iterate all `p.hero-profile__tag` elements
   - Check any tag for "title holder" (case-insensitive) → `isChampion = true`
   - Find first tag matching `/^#(\d+)\s+(.+)$/` → `rankingPosition = parseInt(match[1])`
   - If champion, set `rankingPosition = undefined`
6. Add fields to `events.service.ts` upsert block

### Phase 2 — Frontend (US2)
1. Add `rankingPosition?: number` and `isChampion?: boolean` to `Fighter` in `api.ts`
2. Create `RankBadge.tsx` component:
   - `isChampion` → gold `C` badge
   - `rankingPosition` → white `#N` badge
   - Neither → renders nothing
3. Add `RankBadge` to `BrowserFightCard.tsx` inline with fighter names
4. Add `RankBadge` to `MobilePicks.tsx` inline with fighter names
5. Update `FightCardShowcase.tsx` mock data to include ranking for visual testing

## Scraper Logic (reference)

```typescript
// In scrapeFighter(), after existing nickname extraction:

const tags = $('p.hero-profile__tag').toArray();
let rankingPosition: number | undefined;
let isChampion: boolean | undefined;

for (const el of tags) {
  const text = $(el).text().trim().replace(/\s+/g, ' ');
  if (text.toLowerCase().includes('title holder')) {
    isChampion = true;
  }
}

if (!isChampion) {
  for (const el of tags) {
    const text = $(el).text().trim().replace(/\s+/g, ' ');
    const match = text.match(/^#(\d+)\s+(.+)$/);
    if (match) {
      rankingPosition = parseInt(match[1], 10);
      break;
    }
  }
}
```

## RankBadge Component (reference)

```tsx
// frontend/src/components/RankBadge.tsx
function RankBadge({ fighter }: { fighter: Fighter }) {
  if (fighter.isChampion) {
    return <span className="text-[11px] font-black text-yellow-400 leading-none">C</span>;
  }
  if (fighter.rankingPosition != null) {
    return <span className="text-[11px] font-black text-white leading-none">#{fighter.rankingPosition}</span>;
  }
  return null;
}
```
